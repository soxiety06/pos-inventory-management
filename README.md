# Ayen Aquatics POS & Inventory System 🐠

A decoupled, Progressive Web App (PWA) designed for seamless Point of Sale transactions and inventory management. 

## Architecture
* **Frontend:** Vanilla HTML, CSS, and JavaScript hosted on GitHub Pages.
* **Backend:** Serverless REST API built with Google Apps Script.
* **Database:** Google Sheets.

## Features
* **PWA Native Install:** Fully installable on Android and iOS devices for a full-screen, app-like experience.
* **Responsive UI:** Tailored for both desktop web browsers and mobile devices.
* **PIN Authentication:** Secure 4-digit PIN access for cashiers and administrators.
* **Real-time Dashboard:** Live analytics, daily revenue tracking, and low-stock alerts.
* **Transaction Management:** Full POS cart functionality with pro-rated discounts, exact change calculation, and voiding capabilities.
